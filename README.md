# Grok-Auto-Saver

**Grok-Auto-Saver** は、xAIの画像生成AI「Grok」で生成・閲覧した画像やプロンプトを、**全自動で保存・整理・管理** するためのオープンソース・ツールセットです。

Chrome拡張機能による「リアルタイム自動収集」と、Pythonスクリプトによる「高度な整理＆ビューアー生成」の2段構えで、あなたの生成ライフを強力にバックアップします。

## 🚀 主な機能

### 1. Chrome拡張機能 (GrokSaver)

ブラウザにインストールするだけで、以下の処理をバックグラウンドで自動実行します。

- **画像自動保存:** Imagine画面やFavorites画面で表示された画像を検知し、即座に保存します。
  - 保存先: `Downloads/Grok-Auto-Saver/_Data/Images/YYYYMMDD/`
- **重複防止:** URLおよびハッシュ値の照合により、同じ画像の二重保存を防止します。
- **高画質リカバリー:** Favoritesからの保存時、サムネイルではなくオリジナルの高画質画像を自動的に取得・保存します。
- **ファイル名の最適化:** 保存エラーや文字化けを回避するため、ファイル名は安全な英数字（URLハッシュ等）に自動変換されます。
- **プロンプト自動保存:** 生成に使用されたプロンプト（呪文）をテキストファイルとして保存します。
- **Favoritesログ記録:** お気に入り（Favorites）画面で閲覧した画像の情報をログとして記録し、ビューアーでの絞り込みを可能にします。
- **動画ダウンロード支援:** 動画ポストを開くと、標準のダウンロード機能をトリガーして保存を支援します。
  - ※動画は一旦「ダウンロード」フォルダ直下に保存され、後の整理ステップで移動されます。

### 2. 整理・ビューアツール (Organizer)

ダウンロードフォルダに溜まったファイルを、ワンクリックで整理・可視化します。

- **動画の自動整理:** ダウンロードフォルダ直下の `grok-video-*.mp4` を検知し、日付フォルダへの移動と日時ベースのリネームを行います。
- **プロンプトの統合:** 逐次保存されたプロンプトファイルを、一つの統合履歴ファイル (`All_Prompts_Merged.txt`) にマージします。
- **スマートアーカイブ:** バックアップ（Archivedフォルダ）を常に最新の1世代のみ保持し、ディスク容量の肥大化を防ぎます。
- **不要ファイルの掃除:** 生成失敗時の極小画像やプロフィール画像などを自動でクリーンアップします。
- **HTMLビューアー生成:** 保存した画像・動画・プロンプトを時系列で閲覧できる `Grok_Viewer.html` を生成し、自動で開きます。
  - **キーワード検索:** 過去のプロンプト内容から作品を検索可能。
  - **Favorites特別表示:** お気に入りや動画を各日付のトップにまとめて表示。
  - **折りたたみ機能:** プロンプト単位で画像一覧の開閉が可能。

## 📦 インストール方法

### ステップ1: フォルダの配置

1. [Grok-Auto-Saver GitHubリポジトリ](https://github.com/urea/Grok-Auto-Saver) にアクセスします。
2. ページ右上にある緑色の **「Code」** ボタンをクリックします。
2. **「Download ZIP」** を選択して、プロジェクト全体をダウンロードします。
3. ダウンロードしたZIPファイルを解凍し、中身のフォルダ一式を **PCの「ダウンロード」フォルダの中** に配置してください。
4. フォルダ名は **`Grok-Auto-Saver`** に変更（リネーム）してください（解凍時は `Grok-Auto-Saver-main` 等になっている場合があります）。

推奨パス: `C:\Users\ユーザー名\Downloads\Grok-Auto-Saver`

### ステップ2: Pythonライブラリの準備

画像整理ツールに使用する `Pillow` をインストールしてください。

```bash
pip install Pillow
```

### ステップ3: Chrome拡張機能の導入

1. Chromeで `chrome://extensions/` を開く。
2. **「デベロッパーモード」** をONにする。
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック。
4. 本ツール内の **`_App/ChromeExtension`** フォルダを選択して読み込む。

## 🎮 使い方

1. **収集:** Chromeで Grok を使い、画像を生成したりタイムラインを眺めます。拡張機能が自動でデータを蓄積します。
2. **整理:** **`_App/Organizer`** フォルダ内の `grok_organizer.py` を実行します。
3. **閲覧:** 自動生成された `Grok_Viewer.html` で、あなたの作品群を振り返ることができます。

## ⚠️ 注意事項

- **ブラウザ設定:** 大量のファイルを保存するため、Chrome設定の「ダウンロード前に各ファイルの保存場所を確認する」を **オフ** にすることを強く推奨します。
  - ※設定方法：Chrome右上の `︙` > `設定` > `ダウンロード` から変更可能です。オンのままだと、保存のたびに確認ダイアログが表示されてしまいます。
- **フォルダ名:** 保存パスを固定しているため、ルートフォルダ名は変更しないでください。
