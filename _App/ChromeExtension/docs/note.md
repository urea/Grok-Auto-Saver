# Grokでの画像生成がもっと楽しくなる！全自動保存＆整理ツール「Grok-Auto-Saver」を作りました

xAIの「Grok」で画像生成を楽しんでいる皆さんに、ぜひ使ってみてほしいツールを開発しました。
その名も、 **Grok-Auto-Saver** 。

「Grokで画像をたくさん作ったけど、保存するのを忘れて流れてしまった……」
「あの画像を作った時のプロンプト、何だっけ？」
「ダウンロードフォルダが画像だらけで整理がつかない！」

そんな悩みを、**「全自動」**で解決するツールセットです。

## 💡 なぜこれを作ったのか

Grokの画像生成は非常に強力で楽しいですが、後から振り返るための管理機能はまだ発展途上です。

手動で右クリック保存を繰り返すのは手間ですし、お気に入り（Favorites）に入れても、いつかタイムラインに埋もれてしまいます。何より、 **「画像」と「プロンプト」を紐付けて管理する** のが意外と大変でした。

そこで、「ブラウザで眺めているだけで勝手に保存され、ワンクリックで綺麗に整理される」仕組みを作りました。

## 🚀 Grok-Auto-Saverでできること

このツールは、 **Chrome拡張機能** と **Pythonスクリプト** の2段構えで動作します。

### 1. 【自動収集】眺めるだけで保存完了

拡張機能をインストールしておけば、Grokの画面を見ている間に表示された画像をバックグラウンドで自動保存します。

- **右クリック保存は不要：**  検知した瞬間、日付フォルダへ自動で振り分けます。
- **プロンプトも逃さない：**  生成に使った「呪文」も一緒に保存。後で「どうやって作ったっけ？」と悩むことがなくなります。
- **動画ダウンロード支援：**  動画ポストを開くと、ダウンロードを強力にアシストします。
### 2. 【自動整理】バラバラのデータを1秒で整理

ダウンロードフォルダに溜まったファイルは、専用の整理ツール（Python）を実行するだけで一瞬で片付きます。

- 保存エラーを防ぐためのファイル名最適化。
- フォルダ直下に散らばった動画を日付フォルダへ自動移動＆リネーム。
- 生成失敗時の不要なファイルの自動クリーンアップ。
### 3. 【閲覧】あなただけの専用ギャラリーが誕生

整理が終わると、自動的に**「Grok Viewer」**というHTMLファイルが生成されます。

- **タイムライン表示：**  「いつ、どんな呪文で、何を作ったか」が時系列で並びます。
- **キーワード検索：**  過去の膨大なプロンプトから、キーワードで作品を探せます。
- **お気に入り抽出：**  お気に入り（Favorites）に入れた作品だけを絞り込んで眺めることができます。
## 📦 導入はかんたん3ステップ

- **フォルダを置く：**  指定のフォルダをダウンロードフォルダに配置。
- **拡張機能を入れる：**  Chromeに専用の拡張機能を読み込む。
- **Pythonを実行：**  整理したい時にスクリプトをダブルクリック。詳しい設定方法は、GitHubのREADMEにまとめています。
